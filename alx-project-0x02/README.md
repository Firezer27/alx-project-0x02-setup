# Next.js Project Setup and Basics

This project is the initial setup for a Next.js application using TypeScript and Tailwind CSS.
It is part of the ALX Frontend specialization and prepares the environment for building reusable components, routing, and API integration.

## Features
- Next.js with TypeScript
- Tailwind CSS for styling
- ESLint for code quality
- Basic project structure created for future tasks

## How to Run the Project

```bash
npm install
npm run dev -- -p 3000
